Preparations needed to run the application successfully:
•	Import the kc_energy.sql file to your MySQL database.
•	If you want to test the Send Email function, please make sure your local host has an SMTP server running, or 	you can run the fakeSMTP-2.0.jar file and start the server.
•	Use your MySQL database credential to login this application
The application can either be run by executing the KC_Energy_v2.jar file located in the Exported Application folder or using Eclipse to open the project files located in the Eclipse Project folder. 
Notice: you might have to reimport the external jar files needed so that the eclipse could run the application.
All external jar files are in the lib folder inside of the KC_Energy project folder.
